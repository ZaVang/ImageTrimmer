# ImageTrimmer
一款具备基础功能的简易图片编辑处理工具，支持调整尺寸、裁剪、旋转、翻转、色彩调整以及导出多种图标格式。

## 功能特点
- 多种操作模式：
    - 缩放 - 按指定尺寸等比例缩放图片
    - 裁剪 - 精确裁剪图片的指定区域
    - 缩放+裁剪 - 先缩放再精确裁剪
- 基础编辑功能：
    - 图像旋转（向左/向右90度）
    - 水平/垂直翻转
    - 亮度、对比度、饱和度调整
- 多种裁剪预设
- 多种输出格式：
    - 常规图像格式：PNG, JPEG, GIF, BMP, TIFF
    - 图标格式：ICO (Windows), ICNS (macOS), PNG图标集
- 便捷操作：
    - 拖放功能支持
    - 预览功能
    - 缩放查看
    - 批量处理能力
## 系统要求
Windows 7/8/10/11 或 macOS 10.12+

显示分辨率 1280x720 或更高

## 使用说明
### 基本操作流程
1. 选择源文件
    - 点击"浏览..."选择图片文件
    - 或直接将图片拖放到应用窗口
2. 设置输出选项
    - 选择目标文件夹
    - 设置新文件名
    - 选择输出格式
3. 选择操作模式
    - 缩放：设置目标宽度和高度
    - 裁剪：显示裁剪框并调整位置和大小
    - 缩放+裁剪：两者结合使用
4. 应用基础编辑（可选）
    - 旋转或翻转图像
    - 调整亮度、对比度和饱和度
5. 预览和保存
    - 点击"预览修改"查看效果
    - 点击"应用更改"确认修改
    - 点击"保存图片"导出结果
